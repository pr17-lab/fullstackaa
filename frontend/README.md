# Student Tracker Frontend

This is the React + TypeScript frontend for the Student Academic Tracker system.

## Prerequisites

- Node.js (v18 or later)
- npm (v9 or later)

## Installation

Since the project was scaffolded manually in an environment without Node.js, you need to install dependencies first:

```bash
cd frontend
npm install
```

## Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

## Features

- **Dashboard**: Real-time analytics, grade distribution, and top performers.
- **Student List**: Paginated grid view with Branch and Semester filtering.
- **Student Profile**: Detailed view with GPA trend charts and academic history.

## Tech Stack

- **Framework**: Vite + React + TypeScript
- **Styling**: Tailwind CSS (custom configuration in index.css)
- **Icons**: Lucide React
- **Charts**: Recharts
- **API**: Axios
- **Routing**: React Router DOM v6
